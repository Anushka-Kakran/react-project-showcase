# 🛒 ShoppyGlobe - E-Commerce Application

ShoppyGlobe is a basic e-commerce web application built using **React (Vite)**, **Redux Toolkit**, and **React Router**.  
It allows users to browse products, search items, view product details, add items to cart, and place orders.

---

## 🚀 Features

### 🏠 Home (Product List)
- Fetch products from API: https://dummyjson.com/products
- Display products in responsive grid
- Search functionality using Redux

### 📦 Product Detail
- View detailed information of selected product
- Dynamic routing using product ID

### 🛒 Cart
- Add products to cart
- Increase / decrease quantity
- Remove items from cart

### 💳 Checkout
- Dummy form for user details
- Place order button
- Clears cart after order
- Redirects to Home page

### ❌ 404 Page
- Displays error message for invalid routes

---

## 🧠 Tech Stack

- React (Vite)
- Redux Toolkit
- React Router (createBrowserRouter)
- Tailwind CSS
- JavaScript (ES6)
